Contribute to Vorlon.JS

Vorlon.JS is an open source project with the goal of making it easier to debug and test complicated JavaScript that is common in today’s modern web. Built by a small team of engineers and technical evangelists at Microsoft, the project welcomes developers who like to make tools that help other developers spend less time debugging and more time building the next great web app.

See our GitHub repo

Here are a few ways to contribute

Take on an issue from our issues list or submit an issue you’re having.

Fix an Issue

Extend Vorlon.JS by creating a new plugin to use and share with the community.

Create a Plugin

If you have an idea for a useful plugin, let us know by creating an issue.
